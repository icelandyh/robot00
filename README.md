# robot00
robot00机器人说明书
note: 很多人担心moltbot的安全性，认为moltbot的权限太大，可能泄密，但是我们为什么一定要把moltbot跟自己的过往绑定呢，
我们完全可以搭载一个权限的自进化成长的moltbot呀，暂且将其命名为rboto00. 以下将详述其产品技术方案

机器人不论是人行机器人和事其他形态的机器人，都可以，
一个全新的机器人搭载moltbot, 所有需要访问的站点都是全新的robot00自己注册，个人可以给robot00发送任何指令，让robot00
去完成（当然是需要给robot00发薪水的）
这样robot00(moltbot)是个人的独立助力，而不是影子(当前所有的安全性讨论都是基于moltbot是个人的影子来讨论，为什么molbot不能是一个独立的全新的个体呢？)
